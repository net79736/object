## 프로젝트 개요

Theater 프로젝트는 객체지향 설계 원칙을 학습하기 위한 프로젝트입니다. 
각 버전(v1 ~ v4)은 점진적으로 개선된 설계를 보여줍니다.

### 주요 학습 내용
- 자율적인 객체 설계
- 책임의 분리
- 다형성과 추상화
- 할인 정책과 할인 조건의 유연한 설계
